セットアップ: GitHub Personal Access Token

このツールは GitHub Gist を使って PC と携帯の間でテキストを共有します。
トークンはこの端末のブラウザ内 (localStorage) にのみ保存され、GitHub以外には送信されません。

  1. https://github.com/settings/tokens を開く
  2. 「Generate new token」→「Generate new token (classic)」を選択
  3. Note: cutandpaste など好きな名前
  4. Expiration: 任意 (No expiration でも可)
  5. Scopes は gist だけにチェック
  6. 「Generate token」を押し、表示された ghp_xxx... をコピー
  7. 下の欄に貼り付けて「保存して接続」

※ PC と携帯で同じトークンを使ってもいいし、それぞれ別のトークンを作っても OK (同じ GitHub アカウントで)。
