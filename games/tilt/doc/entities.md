# MOMO Tilt — エンティティ一覧

## アイテム

| アイコン | 種類 | 効果 |
|---------|------|------|
| ❤ | LIFE | ライフ +1 |
| ⭐ | SCORE | スコア +200 |
| ❄ | FREEZE | 取得したボール以外の全ボールを10秒間凍結 |
| 🛡 | SHIELD | 全ボールを5秒間無敵にする（モンスターがいるステージのみ出現） |

## 敵キャラクター

| アイコン | 種類 | 移動 | 攻撃 |
|---------|------|------|------|
| 👻 | PATROL（パトロール） | ランダムに移動、壁で反射 | 当たるとライフ -1 & スタートへ戻る |
| 🔴 | TRACKER（追跡） | 最も近いボールを追跡 | 当たるとライフ -1 & スタートへ戻る |
| 🔵 | SLOW（スロー） | ゆっくり動く | 当たってもライフ減なし（スロー効果のみ） |

## その他オブジェクト

| アイコン | 種類 | 効果 |
|---------|------|------|
| ⚫ | PIT（ホール） | 落ちるとスタートへ戻る（ライフ減なし） |
| 🔑 | KEY（鍵） | ボールの色で表示、番号が一致する錠前ゴールを解錠する |
| 🔒 | LOCKED GOAL（錠前ゴール） | 対応する鍵を取るまで入れない |

## ボール

- 各ステージに 1〜5 個（ステージ番号 = ボール数、最大5）
- 色：赤 / 青 / 緑 / 黄 / 紫
- 同じ番号の色のゴールへ転がして同時にクリアを目指す

## スコア

| 条件 | 得点 |
|------|------|
| ボールがゴールに入る | +100 |
| 鍵を取る | +50 |
| ステージクリア | +500 + ステージ番号×100 |
| スコアアイテム取得 | +200 |
| 2000点ごと | ライフ +1 |
