## 1. 製品概要

- **名称**: MOMO Links（タブ式ブックマーク管理ツール）
- **形態**: 単一HTMLファイルで動作するWebアプリ（サーバー不要）。`index.html` ＋ 同フォルダの `momo_sync.js`（Google Drive同期）＋初期データ `bookmark.json`。
- **公開URL**: `https://qiqiroon.github.io/momo/tools/links/`
- **対応環境**: PC／スマートフォンの主要ブラウザ（Chrome / Safari 等）。`file://` での直接起動も可（同期機能のみ無効）。
- **データ保存**: 端末内（localStorage）。任意で Google Drive 同期により複数端末で共有。

## 3. 機能仕様

### 3.1 ブックマークの管理

| 操作 | 仕様 |
|---|---|
| 追加 | URL必須。タイトル未入力時はURLからタイトルを自動取得（→3.7）。同一URLは重複登録不可。同一ドメインの2件目以降は「グループ化するか」を確認。 |
| 編集 | URL／タイトル／コメント／タグ／グループ化を変更可。URLを変更しタイトルが空のときのみタイトルを再取得。 |
| 削除（ゴミ箱へ） | 「ゴミ箱」へ移動（ソフト削除）。一覧から消えるが復元可能。 |
| 完全削除 | ゴミ箱から完全に削除（復元不可）。 |
| 復元 | ゴミ箱から一覧へ戻す。 |

各ブックマークは URL／タイトル／タグ／コメント／登録日を持つ。

### 3.2 表示モード

| モード | 内容 |
|---|---|
| カード | 1件1カード。ファビコン＋タイトル＋フルURL（読み取り専用・選択コピー可）＋タグ＋日付＋コメント。 |
| リスト | 1件1行に圧縮。タイトル＋ホスト＋タグ（最大3）＋日付。行クリックで詳細展開。 |
| グリッド | アイコンを格子状に多数並べる。クリックで該当サイトを開く。ホバー／ラベルでタイトル表示。 |

ドメイン単位の**グループ折り畳み**に対応（グループ化したブックマークはドメインごとに見出しでまとめ、開閉できる）。一覧は100件ずつの遅延表示（スクロールで追加）。

### 3.3 タグ

- **階層タグ**: `親/子/孫` 形式でフォルダのように多段管理。一覧では末端名のみ表示（同名末端が複数あるときは全パス表示）。
- **絞り込み**: 複数タグ選択で AND（すべて含む）／OR（いずれか含む）切替。子タグも対象。
- **タグ操作**: 追加、（子ごと）削除。削除は同期にも「削除済み」として伝わり復活しない。
- **並べ替え**: 名前順／件数順。

### 3.4 検索

検索欄の文字列で**タイトル・URL・タグ・コメント**を対象に大文字小文字を区別せず即時フィルタ。

### 3.5 インポート / エクスポート

| 種別 | 形式 | 用途 |
|---|---|---|
| エクスポート（JSON） | `bookmark.json`（後述4章） | バックアップ・端末移行 |
| エクスポート（Chrome用HTML） | Netscape Bookmark 形式（タグ＝フォルダ） | 他ブラウザへの取り込み |
| インポート | `bookmark.json`（追記モード・重複URLはスキップ） | 取り込み。旧版形式も自動変換。 |

### 3.6 ファビコン表示

各ブックマークにサイトのアイコン（ファビコン）を表示する。

- **基本方針**: そのページ自身のアイコンを最優先で探し、見つからなければ Google のアイコン問い合わせ→ページ内指定の解析、と段階的に探す。**どうしても無い場合は「地球」を表示する（黒い四角や空白にはしない）**。
- **記憶**: 一度うまく表示できたアイコンの「場所（URL）」をフォルダ単位で端末に記憶し、次回から即表示する（画像そのものは保存しない）。
- **自動格上げ**: 一時的な失敗で不利なアイコン（地球等）を記憶してしまっても、次回表示時に裏で本来のアイコンを取り直し、取れたら自動で差し替え・記憶を更新する。
- **手動リセット**: 「データ管理」から記憶を全消去できる（ブックマークは消えない）。
- **ローカルファイル（file:）**: アイコン取得不可のため、フォルダ＝黄色いフォルダ絵／ファイル＝灰色の書類絵で表示。

優先順位の詳細は[内部仕様書 4章](MOMO_Links_内部仕様書_v4.0.md)を参照。

### 3.7 URLからのタイトル自動取得

追加・編集でURL欄からフォーカスが外れた時、中継（プロキシ）経由でページの`<title>`を取得し、タイトル欄が空なら補完する。ユーザーがタイトル欄に触れると中断。失敗時は無表示でスキップ。**シークレットモード中は外部送信しない**。

### 3.8 シークレットモード

- **目的**: 人に見せたくないブックマーク（タグ `_secret`）を、通常時は隠し、認証時のみ表示する。
- **起動**: ①ショートカットキー（既定 Ctrl+Shift+S、変更可）②ロゴを1.5秒以内に3回タップ ③検索欄に `:secret` 入力。
- **認証**: 初回はPIN設定（2回入力）、以降はPIN入力。**5分間無操作で自動解除**。
- **挙動**: `_secret` タグのリンクのみ表示、画面の枠をオレンジ化＋「🔒 SECRET」表示。シークレット中はタイトル／ファビコンの外部取得を停止。
- **設定**: シークレット中のみPIN・ショートカットキーを変更可。

### 3.9 言語切替

日本語 / English / 中文 / **CAT（猫語）** をドロップダウンで切替。CATは全UIをランダムな鳴き声に置換（エラー系・処理中系・通常系で語彙を変える）。CAT選択直前の言語を基準（catBase）として一部表示を保持。

### 3.10 Google Drive 同期（任意）

複数端末でブックマーク／タグを共有する（HTTPS時のみ。`file://` では無効）。詳細は4.3／[内部仕様書5章](MOMO_Links_内部仕様書_v4.0.md)。

- ON/OFFを「データ管理」で設定（再起動後も保持）。
- 起動時および常駐中（1時間ごとに点検）、前回同期から24時間でバックグラウンド自動同期。自動でできない状態（要再認証等）のときは「●同期が必要」表示が出て、押すと手動同期。
- マージは「更新時刻が新しい方を採用（Last Write Wins）」。削除・タグ削除は「削除済み」記録で伝播。
- 破損対策として、書き込み前世代のバックアップと整合性チェック（チェックサム）を持つ。

### 3.11 ヘルプ

「？」ボタンで基本操作・シークレット・ショートカット・バージョンを多言語表示。

## 4. データ入出力形式

### 4.1 エクスポートJSON（bookmark.json）

```
{
  "version": 2,
  "exported": "<ISO日時>",
  "link_data": [ <ブックマーク> ... ],
  "tags": [ "<タグ>", ... ],
  "tagMeta": { "<タグ>": { "t": <更新時刻>, "del": 0|1 } }
}
```

ブックマーク1件の項目：

| 項目 | 型 | 内容 |
|---|---|---|
| id | 数値 | 一意ID（生成時の時刻ミリ秒） |
| url | 文字列 | 正規化済みURL |
| title | 文字列 | タイトル |
| tags | 文字列配列 | 付与タグ（階層は `/` 区切り） |
| date | 文字列(ISO) | 登録日（表示用） |
| comment | 文字列 | コメント |
| isGrouped | 真偽 | ドメイングループ化 |
| deleted_at | 数値(秒)\|null | 削除時刻（null＝生存） |
| updated_at | 数値(秒) | 最終更新時刻（同期の勝敗判定） |

### 4.2 Google Drive 同期データ（links_data.json）

```
{
  "links":    [ <ブックマーク> ... ],
  "tags":     [ "<タグ>", ... ],
  "tagMeta":  { "<タグ>": {"t":<秒>, "del":0|1} },
  "_checksum":"<SHA-256>",
  "_count":   { "links": <数>, "tags": <数> }
}
```

保存先：Google Drive `/momo-works/links/links_data.json`（前世代を `links_data.bak.json` に退避）。

### 4.3 Chrome用HTMLエクスポート

Netscape Bookmark 形式。タグ＝`<H3>`フォルダ、リンク＝`<A HREF>`。Chrome / Firefox / Safari でインポート可能。

## 5. 非機能・制約

- サーバー不要・完全クライアント動作。データは端末内が基本、同期は任意。
- 外部依存：Google ファビコン／中継プロキシ（タイトル・ファビコンHTML解析）／Google Drive（同期）／Pyodide・GSI（同期実行時に遅延ロード）。
- ブラウザのセキュリティ制約により、他サイトのファビコン**画像の中身は読めない**（場所＝URLのみ記憶）。
- `robots.txt` 等はサブパス公開のため対象外。
