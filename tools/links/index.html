MOMO Links について

MOMO Links は、ブラウザだけで使えるタブ式のブックマーク管理ツールです。複数のフォルダ（タグ）を並べて整理でき、保存したデータはお使いの端末内に残るため、アカウント登録もインストールも必要ありません。ブックマークのエクスポート／インポートや、Chrome 用ブックマーク HTML の書き出しにも対応しています。
