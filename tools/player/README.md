# MOMO works - Personal Project Space


## 日本語

### 概要
ローカルの音楽フォルダをブラウザで再生するスタンドアロンの音楽プレーヤー。サーバー不要、単一HTMLファイル（+style.css）をブラウザで開くだけで動作し、MP3/FLAC/WAV/OGG/Opus等の複数フォーマットに対応、歌詞表示・EQ・タイマー再生など多彩な機能を備えたHDD向け音楽プレーヤーです。

## English

### About
A standalone music player that plays local music folders in your browser — no server required, just open the single HTML file (+ style.css) in your browser. Supports multiple formats including MP3, FLAC, WAV, OGG, and Opus, with features such as lyrics display, equalizer, and timer playback, designed for HDD-bound music libraries.
